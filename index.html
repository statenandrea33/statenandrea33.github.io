GIS in the Community

  • Partnered with Emerald Academy, a K-8 charter school. Aim of the project was to utilize GIS tools to visualize school bus routes for the use of parents.

Overdose Death Analysis

  • Analyzed the amount of overdose deaths in Knox County and its eight surrounding counties.

  • Overdose Death Analysis StoryMap

Python Package

  • Built the package using several open-source packages, including: folium, ipyleaflet, ipywidgets, and geopandas. Package information all hosted on GitHub and was created within VScode.
  • GitHub Repo for Python Package
